# Solution

## Pre-requisites
```
* Ruby version > 2
* RubyGem > 2 
* Rails > 5
* Bundler > 1
* MySQL >= 5.7
```

## Setting up & running the application

The following will seed the csv files from db/seed_data/<FILENAME>.csv files into MySQL database

```
* bundle install
* rails db:setup
* rails s
```
